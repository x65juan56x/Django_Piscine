Ex01: Template engine.

Django templates use special syntax:

  • Blocks: Allow inheritance.
  • Loops: Iterate over lists using {% verbatim %}{% for item in list %}{% endverbatim %}.
  • If structures: Logic control using {% verbatim %}{% if condition %}{% endverbatim %}.
  • Context variables: Display data from views using {% verbatim %}{{ variable }}{% endverbatim %}.
